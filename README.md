# CoreNumero

An advanced numerology web application that blends ancient Pythagorean calculations with modern Gemini AI to provide deep insights into your life path, destiny, and personality.

**What It Does**

- **Numerology Calculations:** Uses Pythagorean numerology methods to compute core numbers like Life Path, Destiny, Soul Urge, and Personality.
- **AI Insights:** Leverages Gemini AI to generate contextual, reflective interpretations and personalized guidance.
- **Interactive Web UI:** Explore charts, reports, and actionable suggestions tailored to your numerology profile.

**Quick Start**

- See the `backend/` and `frontend/` folders for server and client code.
- Start the backend and frontend dev servers according to their README files to run the app locally.
